# React Learning Projects

A collection of React learning projects exploring core concepts and building practical applications.

## Projects

### 1. **learning-react-hooks**
Hands-on practice with React Hooks and state management using `useState`.

**Features:**
- Click Counter
- Text Mirror
- Toggle Button
- Login Toggle
- Character Counter
- Text Visibility
- Add Items

### 2. **learning-todo-app**
A functional todo application with add, delete, and manage tasks functionality.

**Features:**
- Add new todos
- Mark todos as complete
- Delete todos
- Persistent task management

### 3. **learning-movie-app**
A movie browser application fetching and displaying movie data.

**Features:**
- Movie card display
- Movie search functionality
- Responsive header
- Dynamic content rendering

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **CSS** - Styling
- **ESLint** - Code quality

## Getting Started

Each project is a standalone Vite application. To run any project:

```bash
cd <project-folder>
npm install
npm run dev
```

## Project Structure

```
React/

└── learning-movie-app/       # Movie browser app
```

## Learning Goals

- Understand React fundamentals
